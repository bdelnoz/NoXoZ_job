## âš™ï¸ Configuration

### Update _config.yml

Make sure your `_config.yml` has the correct URL:

```yaml
url: "https://bdelnoz.github.io"
baseurl: ""
```

### Customize Content

- **Personal info**: Update email, phone, location in `contact.md`
- **Projects**: Add/remove projects in `projects.md`
- **Experience**: Modify timeline in `experience.md`
- **Skills**: Update tech stack in `about.md`
- **Colors**: Modify CSS variables in `style.css` (`:root` section)

## ğŸ¨ Customization

### Colors

Edit the CSS variables in `style.css`:

```css
:root {
  --primary: #58a6ff;           /* Main blue color */
  --primary-hover: #8bb5ff;     /* Hover blue */
  --bg-dark: #000000;           /* Background */
  --text-primary: #ffffff;      /* Headings */
  --text-secondary: #e0e0e0;    /* Body text */
}
```

### Fonts

The site uses Inter font from Google Fonts. To change:

1. Update the `<link>` in `_layouts/default.html`
2. Update `font-family` in `style.css`

### Adding Pages

1. Create a new `.md` file (e.g., `blog.md`)
2. Add YAML front matter:
   ```yaml
   ---
   layout: default
   title: Blog
   ---
   ```
3. Add link in `_layouts/default.html` navigation

## ğŸ“Š GitHub Pages Deployment

### Automatic Deployment

GitHub Pages automatically rebuilds your site when you push to `main` branch.

### Build Time

- First build: ~2-3 minutes
- Subsequent builds: ~30-60 seconds

### Troubleshooting

**Site not updating?**

```bash
# Force rebuild with empty commit
git commit --allow-empty -m "Trigger rebuild"
git push
```

**404 errors?**

- Check that `_config.yml` has correct `baseurl` (should be empty for `username.github.io`)
- Verify all links start with `/` (e.g., `/about` not `about`)

**CSS not loading?**

- Ensure `style.css` is in root directory
- Check browser console for errors (F12)
- Try hard refresh (Ctrl+Shift+R)
